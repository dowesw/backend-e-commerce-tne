# usage 
## initialise
- clone projet
- execute in cmd : npm install
- create file .env with content :
    * MONGODB_USER=...
    * MONGODB_PASSWORD=...
    * MONGODB_DATABASE=...
    * MONGODB_CLUSTER=...

    * JWT_SECRET_KEY=...

    * MONGODB_URI_PROD=...
    * MONGODB_URI_TEST=...
## run
npm run dev
## test
npm run test
